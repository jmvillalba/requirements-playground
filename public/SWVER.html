1.0 Software unit verification

This document outlines the verification tests at the unit level.

1.1 Localization

1.1.1 Test publishing SWVER0001

This test shall verify SWRS0001 by doing x.

This also shows linkage to cpp files, optionally also with line numbers:

tests/test.cpp (line 29)

⇡ Parents: SWRS0001 Publish frequency

1.1.2 Test publishing SWVER0002

This test shall verify SWRS0001 and SWRS0002 by doing y.

The test is to be conducted as follows:

  • Set up test rig
  • Execute the test
  • Record the results and store at z

⇡ Parents: SWRS0001 Publish frequency, SWRS0002 Another thing
